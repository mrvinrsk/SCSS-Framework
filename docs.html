Accordions

Accordions are simple boxes with a summary and a longer text within it. When you click on an accordion it will open up and show the insides information. Try it out!

You're able to tint accordions by simply adding accordion-COLOR on top of the accordion class as seen as in the example below. COLOR has to be a color from Upgrades colorscheme.
